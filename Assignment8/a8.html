Final Project Proposal

My end goal is to make a game that is very replayable due to random/procedural generation.

My idea for the final is to make a 2D side scrolling infinite runner game, where the player has to move a character around using a mouse. Also, as a fun challenge, I want to make all of the visuals using only CSS. There will be a lot of background elements generated using JavaScript and CSS, like clouds and trees for example. There should be a point system showing the distance the player has traveled.

The playerâ€™s speed will increase as time goes on, making the game harder, but also increasing the rate at which points are gained. There should be a good variety of obstacles, like walls moving horizontally, maybe some projectiles dropping from above, or lasers shooting from the side. The goal is to make sure the player is looking in all directions. All of the visuals for these elements are probably going to be made through pure CSS.

Some of the obstacles I'm thinking of including are:

  • Walls that move left
  • Lasers that shoot horizontally across the screen
  • Alien ships at the top of the screen with lasers that toggle on and off
  • Fireworks shooting from the bottom
  • Not an obstacle, but maybe an item that gives the player an extra life

The only library I am planning on using is JQuery, mainly for tracking mouse movement, but I might find other uses for it down the line.
